Quick tip for editing course content.

You can edit content by doing the following:

  1. Identify the content resource you want to edit.
  2. Create a resource within the concerned module and upload the content.
  3. Reorder the resource, so it goes to the correct position within the course.
  4. Delete the old resource that you no longer need.
  5. Save your changes as and when needed.
  6. Once you are done with all changes, click "Send for review".

Note: Till your edits are reviewed and published, your changes will not take effect. Your current version of the course will continue to be available to the learners on the platform.
